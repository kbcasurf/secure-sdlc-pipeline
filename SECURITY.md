# Security Policy

## ⚠️ This repository is intentionally vulnerable

The application under [`app/`](app/) contains **deliberately planted security
vulnerabilities** used to demonstrate the DevSecOps pipeline in this repo. They
are documented in [`docs/findings-walkthrough.md`](docs/findings-walkthrough.md).

**Do not deploy this application, and do not treat its planted flaws as
disclosable vulnerabilities** — they are the point of the demo.

## Reporting a real issue

If you find a security problem in the **pipeline, tooling configuration, or docs**
(as opposed to the intentionally vulnerable demo app), please open a private
report via GitHub Security Advisories, or contact the maintainer. Please do not
open a public issue for a genuine vulnerability.
